NIE  → 
SIETE NÚMEROS DC
El NIE está formado por una letra inicial (X,Y ó Z) seguida de siete números y otra letra final (dígito de control). El NIF (Número de Identificación Fiscal) para las personas físicas con NIE es igual a este último.
